Hey there! This is the store locator. With a map!

  • You'll need to sign up/sign in to access the content. Click Sign up to get started or Sign in to use a Google/Twitter/GitHub account.
  • When you're all signed in, you'll be able to access the stores data by going into the Stores menu!

This app was created from the MEAN.JS generator... so there's a lot of goodies

  • Map of stores with HeatMap based on the number of visitors. (It's sloooow. Sorry about that. Angular and Google Maps have trouble handling more 1000 markers... There are a few performance tweaks that could be used, like marker clustering, but that's for another time!)
  • Data is stored in a local mongo database running on the same EC2 instance as the app.
  • You can list/create/edit stores directly in the web app or use the API. Here is the POSTMAN collection
  • User login using Passport strategies -> Google/Twitter/GitHub or "local user". Heck, I'll even send you an email to reset your password if you need to!
